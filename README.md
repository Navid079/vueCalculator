# Vue Calculator

A simple calculator in Vue.js

## Summary

Vue Calculator is a simple calculator using Vue.js Framework by [Navid079](https://www.github.com/Navid079)  
Features:
- Addition
- Subtraction
- Multiplication
- Division
- Percent
- Memory
- Dark Mode

Note that the calculator respects the "Order of Operators".
